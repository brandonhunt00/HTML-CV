# Currículo em HTML Básico

Este projeto consiste na criação de um currículo em HTML básico, desenvolvido como atividade acadêmica.

## Objetivo

Construir uma página HTML que apresente informações pessoais e profissionais de forma organizada, utilizando elementos básicos da linguagem HTML.

## Requisitos do projeto

O currículo foi desenvolvido com os seguintes elementos:

- Dois frames (`iframe`), sendo um com um vídeo de apresentação
- Três URLs para páginas favoritas
- Uma lista ordenada com três hobbies principais
- Uma lista não ordenada com quatro soft skills
- Uma tabela com histórico profissional
- Uma foto de perfil
- Organização do conteúdo em uma grande tabela sem bordas visíveis

## Estrutura do projeto

    .
    ├── cv.html
    ├── foto.jpeg
    └── README.md

## Como executar

Clone este repositório:

    git clone <url-do-repositorio>

Acesse a pasta do projeto:

    cd <nome-da-pasta>

Abra o arquivo `cv.html` no navegador.

No macOS, você pode usar:

    open cv.html

Ou simplesmente dar duplo clique no arquivo `cv.html`.

## Tecnologias utilizadas

- HTML5

## Conteúdo da página

A página contém:

- Dados pessoais
- Resumo profissional
- Idiomas
- Hobbies
- Soft skills
- Links favoritos
- Histórico profissional
- Foto
- Vídeo de apresentação

## Observações

- A imagem de perfil deve estar na mesma pasta do arquivo HTML.
- O nome esperado da imagem é `foto.jpeg`.
- Para o vídeo funcionar corretamente, é necessário substituir o link de exemplo pelo link real incorporado do YouTube no código HTML.
